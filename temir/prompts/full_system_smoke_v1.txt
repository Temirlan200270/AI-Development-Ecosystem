Создай production-ready Python CLI утилиту.

Функционал:
- читает текстовый файл
- считает:
  1) количество строк
  2) количество слов
  3) количество уникальных слов (case-insensitive)
- сортирует строки по Unicode (корректно для UTF-8)
- выводит результат в JSON

Требования:
- argparse CLI
- encoding=utf-8
- обработка пустых строк
- O(n) решение для подсчёта слов
- структура проекта (относительно output_dir): подпапка cli_tool/ с файлами main.py, utils.py и каталогом tests/ с test_cli.py (pytest)
- простые unit-тесты (pytest)

Выход:
- готовый рабочий код в cli_tool/
- тесты
- никаких TODO
