# Решение проблемы с diff_match_patch

## Проблема
```
ModuleNotFoundError: No module named 'diff_match_patch'
```

## Причина
Python из Windows Store часто имеет ограничения и может не работать корректно с pip.

## Решения (попробуйте по порядку)

### Решение 1: Установка через pip напрямую

Откройте PowerShell или CMD и выполните:

```powershell
pip install diff-match-patch
```

Если не работает, попробуйте:

```powershell
python -m pip install diff-match-patch
```

или

```powershell
py -m pip install diff-match-patch
```

### Решение 2: Установка всех зависимостей

```powershell
pip install -r requirements.txt
```

### Решение 3: Если Python из Windows Store не работает

1. **Скачайте Python с официального сайта:**
   - Перейдите на https://www.python.org/downloads/
   - Скачайте последнюю версию Python 3.11+
   - При установке **ОБЯЗАТЕЛЬНО** отметьте "Add Python to PATH"

2. **После установки откройте НОВЫЙ терминал** и выполните:
   ```powershell
   pip install diff-match-patch
   ```

3. **Проверьте установку:**
   ```powershell
   python -c "import diff_match_patch; print('OK')"
   ```

### Решение 4: Использование виртуального окружения (рекомендуется)

```powershell
# Создайте виртуальное окружение
python -m venv venv

# Активируйте его
.\venv\Scripts\Activate.ps1

# Установите зависимости
pip install -r requirements.txt

# Или только diff-match-patch
pip install diff-match-patch
```

## Быстрая проверка

После установки проверьте:

```powershell
python -c "import diff_match_patch; print('✅ Модуль установлен!')"
```

Если выводит "✅ Модуль установлен!", значит всё готово!
